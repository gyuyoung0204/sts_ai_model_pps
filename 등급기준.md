# PPS 등급 기준 (열처리 · 스프링 · 기타)

PPS(Patrol Priority Scorer)가 불량 건에 **등급·점수**를 매기는 기준을 정리한 문서입니다.
모든 값은 `patrol_lib.py` 상단 상수와 `score()` / `grade_for()` 로직에 그대로 대응합니다.

---

## 1. 점수 계산 (모든 공정군 공통)

기본 **10점**에서 시작해, 아래 요인들을 **곱셈**으로 적용합니다.

| 요인 | 조건 | 가중치 |
|---|---|---|
| 유형 위험도 | 불량 유형별 (아래 3번) | ×0.5 ~ 2.55 |
| 심함 표기 | 불량내용에 "심함" 포함 | ×2.2 |
| 단기 급증 | 같은 위치 최근 집중도(intensity) ≥ 2.0 | ×2.1 |
| 단기 빈발 | intensity ≥ 1.2 | ×1.7 |
| 최근 재발 | intensity ≥ 0.6 | ×1.35 |
| 근래·분산 | intensity ≥ 0.15 | ×1.12 |
| 과거 심함 이력 | 그 위치에 과거 심함 발생 | ×1.2 |
| 장기 누적 | 같은 위치 총 6회 이상 | ×1.1 |
| 다발 설비 | 설비 누적 15건 이상 | ×1.3 |
| 조치요청 문구 | "지시·교체·확인·요망" 등 | ×1.25 |

> **단기 집중도(intensity)** 는 같은 설비·위치의 과거 발생을 시간감쇠(반감기 **10일**)로
> 합산한 값입니다. 전체 누적 횟수보다 **최근에 몰린 발생**을 크게 반영합니다.
> (예: 최근 2주에 3회 ≫ 3개월에 걸쳐 3회)
> 위치 이력은 항상 **해당 건의 발생 시점 이전**만 집계합니다(미래 누수 방지).

## 2. 등급 결정 — 공정군별

### 열처리 (1차열처리 · 2차열처리)
점수 컷으로 등급 결정.

| 등급 | 기준 |
|---|---|
| 🔴 긴급 | 점수 ≥ 60 |
| 🟡 주의 | 점수 ≥ 25 |
| 🟢 일반 | 점수 < 25 |

- 주력 유형은 **가시발생**(유형가중 1.94). 가시발생 + 심함이면 보통 긴급.
- 보빈 위치의 단기 빈발이 점수를 크게 좌우.

### 스프링 (스프링신선 · 스프링습식)
**지정 3개 유형은 점수와 무관하게 등급 고정**, 나머지는 점수 컷.

| 유형 | 등급 | 비고 |
|---|---|---|
| 와이어 이탈 | 🔴 긴급 | 유형으로 고정 (정렬용 가중 2.0) |
| 와이어 튐 | 🟡 주의 | 유형으로 고정 (1.8) |
| 다이스 마모 | 🟢 일반 | 유형으로 고정 (1.6) |
| 그 외 (윤활제·권취·선떨림·냉각 등) | 점수 컷 | 긴급 ≥60 / 주의 ≥25 |

> 정렬은 **등급 우선(긴급→주의→일반) → 점수순**. 와이어 이탈은 점수가 낮아도 항상 최상단.

### 기타 (1차신선 · 단두신선)
점수 컷으로 등급 결정 (열처리와 동일: 긴급 ≥60 / 주의 ≥25 / 일반 <25).
- 기록성·윤활제 위주라 대부분 일반. 심함·스크래치만 점수로 상향.

## 3. 공정군별 주요 유형 가중치 (학습값)

| 공정군 | 유형 | 건수 | 유형가중 |
|---|---|---|---|
| 열처리 | 가시발생 | 90 | **1.94** |
| 열처리 | 탈지불량 | 8 | 0.50 |
| 열처리 | 와이어 이탈 | 11 | 0.50 |
| 스프링 | 다이스 마모 | 32 | 유형고정 → 일반 |
| 스프링 | 와이어 튐 | 21 | 유형고정 → 주의 |
| 스프링 | 와이어 이탈 | 12 | 유형고정 → 긴급 |
| 스프링 | 윤활제 관리 | 11 | 0.50 |
| 기타 | 기록 누락/오류 | 3 | 0.62 |
| 기타 | 윤활제 관리 | 3 | 0.50 |
| 공통 | 선떨림 · 스크래치 | 소수 | **2.55** (표본 부족 과대) |

> 유형가중치는 과거 "심함" 발생률을 베이지안 평활한 값(글로벌 대비 배수, 0.5~3.0 범위).
> 표본이 적은 유형은 과거 심함 1건만 섞여도 가중이 부풀 수 있음 → 조정 가능.

## 4. 조정 가능한 값 (patrol_lib.py)

| 상수/항목 | 위치 | 의미 |
|---|---|---|
| `SPRING_TYPE_GRADE` | 모듈 상단 | 스프링 유형→등급 고정 매핑 |
| `SPRING_TYPE_WEIGHT` | 모듈 상단 | 스프링 지정유형 정렬용 가중 |
| `grade_for(... urgent=60, warn=25)` | 함수 | 등급 컷 점수 |
| `burst_half_life_days=10` | `train_kb()` | 단기 빈발 반감기(작을수록 최근 집중을 더 크게) |
| 단계별 ×배수 | `score()` | 심함·단기빈발·다발설비·조치요청 가중 |

값을 바꾼 뒤에는 `train_model.py`로 재학습하면 반영됩니다.
