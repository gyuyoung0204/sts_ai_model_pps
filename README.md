# PPS — Patrol Priority Scorer

스테인리스 선재 공장의 **공정패트롤 불량 데이터**를 학습해, 새로 발생한 불량 건에
**우선순위 등급·코드·점수·사유**를 자동으로 부여하는 예측 모델입니다.

블랙박스 신경망이 아니라, 과거 데이터에서 패턴(위치별 발생이력·재발성, 유형 위험도,
설비 다발도)을 추출해 투명하게 점수를 매기는 **규칙+통계 기반 스코어링 모델**입니다.
모든 예측에 "왜 그 등급인지" 사유가 함께 나와 현장에서 검증할 수 있습니다.

> ⚠️ 이 저장소에는 코드만 포함됩니다. 실제 공장 데이터·개인정보는 포함되지 않습니다.
> 모델은 `train_model.py`로 직접 보유한 데이터를 학습시켜 사용하세요.

## 동작 구조

```
[학습] 누적 데이터 ──train_model.py──▶ knowledge_base.json (모델)
[예측] 새 데이터  ──predict_new.py──▶ 등급·코드·점수·사유
```

- **학습(매달/누적)**: 지금까지 쌓인 전체 데이터로 모델을 갱신
- **예측(그달 신규)**: 학습된 모델이 새 데이터에만 등급을 부여 (새 데이터는 학습에 쓰지 않음 → 순수 예측)

## 설치

```bash
pip install -r requirements.txt
```

## 사용법

### 1) 모델 학습
```bash
python train_model.py 누적_전체데이터.xlsx
# 여러 달 파일을 함께: python train_model.py 3월.xlsx 4월.xlsx 5월.xlsx
```
→ `knowledge_base.json` 생성/갱신

### 2) 새 데이터 예측
```bash
python predict_new.py 그달_신규데이터.xlsx
```
→ `*_예측결과.csv` (MES 인입용) + `*_예측결과.xlsx` (색상 검토용)

### 3) 대시보드 (엑셀 업로드 → 예측)
```bash
streamlit run dashboard.py
```
→ 브라우저에서 엑셀 업로드, 등급 분포·우선 리스트·필터·다운로드, 등급 컷 실시간 조정

### 4) 코드에서 직접 호출 (DB 연동용)
```python
import patrol_api as API
result = API.score_records([
    {'불량처리 일시': '2026-07-03 09:10', '공정': '1차열처리',
     '설비': '8M-7호', '불량내용': '15 가시발생심함'}
])
# → [{... '공정군':'열처리', '등급':'긴급', '등급코드':1, '점수':113.1, '사유':'...'}]
```

## 입력 / 출력

**필수 입력(건당 4개)**: `불량처리 일시` · `공정` · `설비` · `불량내용`
(선택: `설비불량유형` · `검사원` · `양호조치일시` · `비고`)

**출력(5개)**: `공정군` · `등급`(긴급/주의/일반) · `등급코드`(1/2/3) · `점수` · `사유`

불량내용 텍스트에서 유형·위치번호(보빈/다이스)·심각도·조치요청을 자동 추출합니다.

## 등급 기준

**공통 점수** — 기본 10점 × 가중치(유형 위험도, 심함 ×2.2, 단기 빈발 ×1.35~2.1,
다발 설비 ×1.3, 조치요청 ×1.25 …). 단기 빈발은 전체 누적 횟수보다 **최근에 몰린 발생**을 크게 반영.

**등급 결정**
- 열처리 · 기타: 점수 컷 (긴급 ≥60 / 주의 ≥25 / 일반 <25)
- 스프링: 유형으로 등급 고정 — 와이어 이탈=긴급, 와이어 튐=주의, 다이스 마모=일반. 그 외는 점수 컷

가중치·등급 컷·유형 매핑은 `patrol_lib.py` 상단 상수로 조정할 수 있습니다.

> 📄 상세 기준은 [등급기준.md](등급기준.md) 참고 — 공정군별 점수표·등급표·유형 가중치 전체 정리.

## 파일 구성

| 파일 | 역할 |
|---|---|
| `patrol_lib.py` | 표준화 + 채점 엔진 (핵심 로직) |
| `patrol_api.py` | 호출 인터페이스 `score_records()` (DB 연동 접점) |
| `train_model.py` | 모델 학습 |
| `predict_new.py` | 새 데이터 예측 |
| `retrain.py` | 폴더 단위 누적 재학습 |
| `score_file.py` | 단일 파일 채점 |
| `mes_pipeline.py` | 학습+채점 통합 실행기 |
| `dashboard.py` | Streamlit 대시보드 |
